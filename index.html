Create Opportunity

Step 1.

  • 1.1    Select ID Class to Containment

  • 1.2    Select a Business Account

  • 1.3    Add the location where the work is performed

  • 1.4    Add the contact (customer POC)

Step 2.

  • 2.1    Add the owner (TechGuard employee who received the Opportunity)

  • 2.2    Add a subject (Firm Name, Building number or location, type of Containment)

  • 2.3    Add an estimated close date (choose 2 month after Quote date if unknown)

  • 2.4    Save (at the top right)

Create Sales Quote

Step 1.

  • 1.1    Create a Sales Quote from Opportunity

  • 1.2    Ensure Sales Quote is selected in dropdown

  • 1.3    Click create and review

Step 2.

  • 2.1    Check all the information that carried over

Step 3.

  • 3.1    Add all sales quote lines (some can be configured)

  • 3.2    Add a description for what you are adding

  • 3.3    Add the quantity, unit price, and amount

Step 4.

  • 4.1    Add your unit cost

  • 4.2    Check the total amount and ensure it is correct

  • 4.3    Save

Specialty (configuration)

Step 1.

  • 1.1     Click the + and add your configurable line

  • 1.2 Click the line then click Configure (1.3)

    • 2.1    Add the dimensions of the aisle

    Step 4.

    • 4.1     Select all the things needed

    • 4.2     Save

    • 4.3     Click Finish

    • 4.4     Close the window
